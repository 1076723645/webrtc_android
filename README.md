#### 一、开源借鉴

android端的实现：https://github.com/ddssingsong/WebRtcDemo （github）

服务器端的实现：https://github.com/ddssingsong/webrtc_server.git （github）

#### 二、本demo实现的功能

1. 支持多人视频
2. 支持一对一视频和语音
3. 可配置服务器，包括wss和stun服务器
4. 移动到library，供项目调用
5. 优化界面
6. 动态权限申请
7. 分离Websocket逻辑，可替换和自定义信令
